Preguntas Escenciales

A continuación presentamos términos muy importantes que nos ayudaran a lo largo del curso:

¿Qué es entonces una base de datos - DB?
Una Base de Datos es una colección de archivos que estan interrelacionados, se busca la menor cantidad de datos repetidos y una cuestion importante de las bases de datos es que los archivos contienen muchas similitudes en sus tipo de registro. Como ejemplo si tenemos un archivo donde se tenga registro de los alumnos de una universidad, cada uno tiene su propio id, nombre, carrera etc.

¿En qué casos es conveniente usar bases de datos? en los casos en los que la información que se va a manipular presenta ciertos archivos que esten interrelacionados y que estan informaciñon va a ser manejada por varios mimebros dentro de una organización. Y de igual forme cuando la cantidad de datos a registrar sea bastante grande.

¿Qué es un sistema de gestión de base de datos - DBMS? Database Management System es una colección de datos interrelacionados junto con una serie de programas con la capacidad de acceder a la colección de datos. Utilizamos DBMS para que podamos extraer y guardar información dentro de la base de datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS. Estos nos ayudan a definir estructuras para el almacenamiento del la información contenida en la base de datos como las herramientas para la gestión de información. Y una de las funciones mas importantes que tienen es de mantener la seguridad de los datos, ya que en cualquier accidente que pueda ocurrir estos son los responsables de regresar la información a su último resplado.
