  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    JavaScript no es un lenguaje de programación compilado, este es integrado a HTML y es interpretadp. Java es un lenguaje de programación orientado a objetos y JavaScript a objetos. De igual forma en JavaScript no podemos hacer uso de la herencia y poliformismo y no existe la declaración de variables, y una diferencia importante es que en Java el codigo seefectua al momento de compilarse y en JavaScript las referencias a objetos se realizan al momento de la ejecución. Se puede decir que comparten algo de sintaxis pero fuera de eso son diferentes.

  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    Estan getFullYear() que nos devuelve el año con 4 digítos, get Month() que nos devuelve el número de mes, getDate() que nos devuelve el día del mes, getHours() que nos devuelve la hora actual en formato 24 hrs, y el método getMinutes() que nos devuelve los minutos.

  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    Tenemos el método concat() que nos permite unir dos o más arreglos, indexOf() el cual busca un elemento dentro del arreglo y devuelve la posición, pop() que eliminar el ultimo elemento del arreglo y devuelce el contenido del mismo, push() el cual añade elementos al arreglo y nos devuelve la nueva longitud y reverse() que nos invierte el orden de los ellemetnos del arreglo.

  • ¿Cómo se declara una variable con alcance local dentro de una función?

    Utilizamos la palabra "let" despúes el nombre de la variable.

  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    El problema es que cualquier función o parte del programa puede tener acceso a ella lo que quiere decir es que su valor puede ser modificado y se puede perder el control de su valor por haberle usado en otra parte del código.

  • ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?

    Match ejecuta una busqueda por una coincidencia en una cadena devuelve un array de infromación o null en caso que no haya coincidencia. Search verifica una coincidencia en una cadena. Replace ejecuta una busqueda por una coincidencia en una cadena, reemplaza la subcadena encontrada con la que esta de reemplazo. Split utiliza una expresión regular para poder cortar a una cadena y lo coloca en un array de subcadenas.
