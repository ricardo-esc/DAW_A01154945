Preguntas

  • ¿Qué es ODBC y para qué es útil?

    Estandar que permite manipular datos de cualquier base de datos independientemente del gestor de base de datos que utilice.

  • ¿Qué es SQL Injection?

    Es una vulnetrabilidad la cual permite que el usuario pueda inyectar como dice el termino, instrucciones SQL con la intención de alterar la base de datos realizando acciones malicias.

  • ¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

    Escapar los caracteres especiales utilizados en las consultas SQL, delimitar los valores de las consultas, siempre estar verificando todo dato que sea introducido por el usuario.
